h1. Contest

Run a contest to select a random person who follows you on Twitter.

h2. Instructions

Create a ~/.twitter.yml file. This file contains your Twitter credentials. For example:

    name: thoughtbot
    password: slainte

*Optionally*, create a ~/.twitter_ineligible.yml. This file contains a list of followers who are not eligible to win the contest, such as employees of your company:

    cpytel
    jyurek
    jankowski
    tsaleh
    croaky
    cssboy
    kevinburg
    joeferris
    mikeburns
    umbrellatoday
    widgetfinger

Run the program:

    rake contest
